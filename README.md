## ğŸš€ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/3a7anton/delve-golden-bites.git
   cd delve-golden-bites
   ```

2. **Install dependencies:**
   ```bash
   bun install
   ```

   ```bash
   bun run dev
   ```

   ```bash
   bun run build
   ```

5. **Preview production build:**
   ```bash
   bun run preview
   ```
